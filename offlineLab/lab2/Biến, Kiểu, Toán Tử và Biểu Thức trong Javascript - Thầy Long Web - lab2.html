Biến, Kiểu, Toán Tử và Biểu Thức trong Javascript

20/03/2020

A. BIẾN VÀ KHAI BÁO BIẾN

  • Biến dùng để lưu trữ dữ liệu. Cú pháp:  var tenbien;
  • Tên của biến có thể gồm chữ và số, nhưng không được bắt đầu bằng số, không chứa khoảng trắng (dấu cách ), dấu câu, ngoại trừ dấu gạch dưới (_) . Ví dụ:   var gia_km;
  • Có thể khai báo nhiều biến trên một dòng.
    Ví dụ:    var x, y, z;
  • Có thể vừa khai báo vừa khởi tạo giá trị cho biến.
    Ví dụ:   var x = 1;

B.  KIỂU DỮ LIỆU CỦA BIẾN

Trong JavaScript, biến có thể có những kiểu dữ liệu sau: kiểu số, kiểu chuỗi, kiểu Boolean, kiểu mảng, kiểu object

Kiểu Số

JavaScript không chia ra kiểu Double, Integer… như các ngôn ngữ khác mà gộp lại thành một kiểu duy nhất là kiểu số

<script>
  var x = 20, y=1.5; 
  alert(x + " " + y);
</script>

Các hàm liên quan đến kiểu số

isNAN() : Xác định xem tham số truyền vào có phải là số hay không. Nếu là số trả về False. Nếu không phải là số, trả về True

 <script>
       var x = prompt("Bạn bao nhiểu tuổi");
       if (isNaN(x)==true){
          alert("Trời ơi! nhập số á");
       }
       else {
          alert("Cảm ơn");
       }
</script>

Các hàm trong thư viện Math: là thư viện toán học có nhiều hàm xử lý dữ liệu kiểu số

Kiểu string (chuỗi ký tự)

String được bao quanh bởi cặp dấu nháy kép.  Ví dụ: var s = “Mẹ ơi”;

Các thuộc tính và hàm trên string

  • Thuộc tính length: đếm số ký tự của chuỗi
<script>
   var s = "Mẹ tôi hai tiếng thân thương"; 
   alert("Số ký tự: " + s.length); //28
</script>
  • Hàm substr: trích 1 chuỗi con, bắt đầu từ 1 vị trí và lấy x ký tự
<script>  
  var c = "Chịu khó mới giỏi";  
  alert(c.substr(5,3)); //khó
</script>
  • Hàm concat: ghép 2 chuỗi
<script>
var m = "Mẹ"; t= " tôi";
var s = m.concat(t);
alert(s); //Mẹ tôi
</script>
  • Hàm toUpperCase
<script>
var m = "Mẹ tôi hai tiếng thân thương";
var t = m.toUpperCase();
alert(t);
</script>
  • Hàm  toLowerCase
<script>
var m= "NgưỜi việT rẤT hiếu khách";
var t= m.toLowerCase();
alert(t); //người việt rất hiếu khách
</script>

Kiểu Boolean

Biến kiểu Boolean có 1 trong hai giá trị là true và false. Các giá trị Boolean thường được sử dụng trong các cấu trúc điều khiển để diễn tả điều kiện.

<script>
  var diem = 9;
  var xeploaigioi = diem>=8; 
  if (xeploaigioi==true) alert("Giỏi quá");
</script>

Mảng

Mảng là kiểu dữ liệu dùng để lưu nhiều giá trị trong 1 biến. Khai báo và gán giá trị cho mảng: 

<script>
   var thucung = new Array();
   thucung[0] = "mèo";
   thucung[1] = "chó";
   thucung[2] = "heo";

   console.log(thucung);
   console.log(thucung[1]);
</script>

hoặc

<script>
   var thucung = new Array("mèo", "chó", "heo");
   console.log(thucung);
   console.log(thucung[1]);
</script>

Giá trị null và undefined

Giá trị Null: Null nghĩa là không gì cả

var x = null; nghĩa là x không chứa 1 giá trị nào cả. var x = null; khác với var x = “”;

Kiểu Undefined: Chưa xác định giá trị

<script>
var x;  alert(x);  // undefined
</script>

D.   HÀM PROMPT

Dùng để lấy dữ liệu do người dùng nhập vào. Hàm trả về giá trị mà người dùng đã nhập. Ví dụ: 

<script>
var ht = prompt("Người ơi! Bạn tên gì đó");
alert ("Chào " + ht);
</script>

 => Chào Tèo

LỆNH LỰA CHỌN ĐƠN

if (điều_kiện)
{
    //Thực hiện
}

//Thực hiện

Ví dụ:

if (x < 350) {
//Code thực hiện khi giá trị biến x nhỏ hơn 350

if (x == y) {

ĐIỀU KIỆN PHỨC TẠP

Dùng toán tử lôgíc để kết hợp nhiều mệnh đề đơn thành một mệnh đề phức tạp hơn

var x = prompt("Nhập vào số b<=100 và >50");
if ((x < 51) || (x > 99)) {
  document.write("<h1>Bạn nhập giá trị: " + x + </h1>");
  document.write("<h2>Giá trị này không đúng yêu cầu</h2>");
}
